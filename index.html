About Me

I'm Tim, a full stack developer with experience creating sophisticated web applications using JavaScript and modern JavaScript frameworks. As I continue to grow as a developer, I hope to assist non-profit organizations better reach their consumer-base, and build dynamic tools to improve efficiency.
